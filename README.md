## Scenario
The project team wants to ensure that P1 incident records created shoul have mandatory fields populated before they are submitted. However, in the event the ionformation required to populate them is not available, the fields shuld not be mandatory. Details as to why the information is missing should be added to the 'Additional comments' field.  

## Objective
Build a functionality into the incident form that prompts an agent other than the Major incident manager to confirm Major incident information is included in the record before it is submitted.
